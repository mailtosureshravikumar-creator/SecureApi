What is Aadhaar OCR?

The Aadhaar Extract API enables secure extraction and verification of identity details from Aadhaar records to confirm an individual identity in a compliant and controlled manner. It is designed for regulated digital onboarding environments and minimizes manual intervention while strengthening trust in verification processes.

What Information Does the API Extract?

  • Full Name (as per Aadhaar)
  • Date of Birth / Year of Birth
  • Gender
  • Aadhaar Status (active / inactive)
  • Masked Aadhaar Number (where applicable)
  • Address details (structured format)
  • Verification status and confidence indicators

How to Interpret the API Results

1. Identity Match Status

  • Verified: Aadhaar details are valid and successfully authenticated.
  • Partially Verified: Minor mismatches detected (e.g., spelling variations).
  • Not Verified: Aadhaar details could not be authenticated.

2. Aadhaar Validity & Activity Status

  • Active: Aadhaar is valid and usable for verification.
  • Inactive / Invalid: Aadhaar is not eligible for authentication.

3. Data Confidence Indicators

  • High confidence: Suitable for straight-through processing.
  • Medium confidence: May require secondary checks.
  • Low confidence: Manual review recommended.

4. Error & Exception Codes

  • Invalid input data
  • Authorization or consent issues
  • Data mismatch
  • Temporary service unavailability

This structured response enables faster resolution and process continuity.
