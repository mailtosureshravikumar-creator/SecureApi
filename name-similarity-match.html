What is Name Similarity Match?

The Name Verification API enables organizations to accurately validate and match an individual or entity name against multiple authoritative and trusted data sources. It resolves spelling variations, formatting differences, abbreviations, and transliteration inconsistencies during onboarding and KYC processes.

By intelligently comparing input names with verified reference records, the API improves identity verification precision, reduces discrepancies, and ensures compliant customer data across systems.

How the Name Verification Process Works

Step 1: Name Input & Normalization

  • Removes extra spaces, special characters, and formatting inconsistencies.
  • Handles initials, abbreviations, and common name variations.
  • Normalizes case sensitivity and phonetic differences.

Step 2: Multi-Source Name Matching

  • Government or regulatory records
  • Financial and identity databases
  • Internal enterprise reference systems

Step 3: Advanced Matching Logic

  • Fuzzy matching algorithms
  • Phonetic similarity checks
  • Sequence and word-order analysis

Advanced Matching Logic
